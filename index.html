Qlearnkit: a python toolkit for Quantum Machine Learning

Welcome to the documentation page for qleanrkit.

Qlearnkit is a python library implementing machine-learning algorithms and models for a gated quantum computer, mainly developed for academic purposes at Institute EURECOM.

If you find any issues, please report it at the GitHub page.
